BrowserPlus Mobile

Before using BrowserPlus, it is your responsibility to read the README.md document on GitHub.

For a more thorough experience, access BrowserPlus, on a desktop, laptop, Mac, etc.

BrowserPlus offers an application for iOS. Simply install the free application using Safari. Go to the "share" icon at the bottom of your screen and find the option for adding the site to your home screen. A shortcut to the app will be installed on your homescreen. Please note that this will not simply save a shortcut to your home screen, but it will also install a web application on your device. It does not take up significant storage.
